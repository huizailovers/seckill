# 秒杀系统
## 技术架构
### 技术选型

>  开发框架  `springboot` + `springmvc` + `mybatis` + `spock`（单测框架）
 
>  中间件   `redis` + `kafka` + `mysql`

>  `TDD` 基于测试驱动开发模式编写代码，所有代码基于spock来做测试
